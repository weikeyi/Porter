# 路线图

> Porter 未来规划与迭代方向

---

## 当前状态 (v0.1.12)

Porter 已具备的能力：

- ✅ 三种配置输入模式（清单文件、嵌套配置、目录）
- ✅ 三种操作（复制、移动、删除）
- ✅ 预检扫描与匹配验证
- ✅ Robocopy 多线程复制引擎
- ✅ 实时进度追踪
- ✅ 三种冲突处理策略
- ✅ 动态子目录发现
- ✅ 操作日志记录
- ✅ Windows Portable 打包

---

## 短期规划

### 通用性增强

- [ ] **文件级复制支持**：当前仅支持按目录名匹配和复制，扩展为支持单个文件级别的批量操作
- [ ] **通配符 / 正则匹配**：清单中支持 `Project_*`、`Data_202?` 等模式匹配
- [ ] **排除规则**：支持在清单中指定需要排除的目录或文件模式
- [ ] **相对路径支持**：清单中的路径允许使用相对路径（相对于源目录）

### 配置增强

- [ ] **配置模板**：保存和加载常用的配置组合（源目录、目标目录、策略）
- [ ] **历史记录**：记录最近使用的配置，快速恢复
- [ ] **CSV 高级解析**：支持多列 CSV，可指定哪一列为目录名、哪一列为目标路径

### 用户体验

- [ ] **批量选择 / 反选**：预检结果中可手动勾选或取消部分目录
- [ ] **暂停 / 恢复**：长时间任务支持暂停和恢复
- [ ] **取消操作**：执行中可安全取消

---

## 中期规划

### 性能优化

- [ ] **增量复制**：基于文件修改时间或 hash 只复制有变化的文件
- [ ] **并行区间处理**：多个区间同时执行复制
- [ ] **网络路径优化**：针对 UNC 路径 / 映射网络驱动器的连接管理和超时配置

### 功能扩展

- [ ] **插件式后端**：可选复制后端（Robocopy / xcopy / 原生 fs），适配不同场景
- [ ] **校验功能**：复制完成后自动对比源和目标的文件 hash（如 MD5/SHA256）
- [ ] **任务队列**：多任务排队执行，支持任务优先级
- [ ] **定时任务**：支持计划执行（如每日凌晨自动备份）

### 界面升级

- [ ] **多标签页**：支持同时配置和执行多个独立任务
- [ ] **暗色/亮色主题切换**
- [ ] **系统托盘**：最小化到系统托盘，后台静默执行

---

## 长期愿景

### 跨平台

- [ ] **macOS / Linux 支持**：使用原生 fs 或 rsync 替代 Robocopy
- [ ] **远程同步**：支持 SFTP / S3 等远程目标

### 自动化

- [ ] **CLI 模式**：支持命令行调用，可集成到脚本和 CI/CD 流程
- [ ] **API 接口**：通过本地 HTTP 接口或 WebSocket 进行远程控制
- [ ] **配置热重载**：监控清单文件变化，自动触发复制
